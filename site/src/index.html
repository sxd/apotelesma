Apotelesma

Static PostgreSQL branch analytics published from the Apotelesma repository.

This site is built from the sxd/apotelesma GitHub repository. CI runs PostgreSQL 18 together with the ekorre FDW at build time, exports JSON snapshots, then publishes a fully static artifact to GitHub Pages.

Published Files

Static JSON and Grafana starter assets

These files are the published inputs behind the page and can be consumed directly from other tools.

branches.json Branch metadata and build settings branch_summary.json Branch-level aggregate metrics commits.json Commit-level records used by the interactive explorer authors.json Author activity export daily_activity.json Daily branch activity data trailer_fields.json Allowed trailer fields and client-facing metadata trailer_summary.json Trailer value summary across allowed fields trailer_people_summary.json People mentioned in person-oriented trailer fields commit_trailers.json One row per extracted trailer occurrence grafana.json Friendly consolidated payload for Grafana and external consumers apotelesma-starter-dashboard.json Starter Grafana dashboard for static JSON via Infinity apotelesma-postgresql-dashboard.json Starter Grafana dashboard for direct PostgreSQL queries
